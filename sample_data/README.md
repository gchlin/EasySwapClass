# sample_data/ — 匿名範例資料

本資料夾含有**完全虛構的匿名課表資料**，用途：

- 測試 `scripts/` 中的腳本是否正常運作
- 了解 CSV 欄位結構與資料格式
- 示範完整操作流程（不需要真實課表 PDF）

**這裡的資料沒有真實老師姓名或課表內容，可安全上傳 GitHub。**

---

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `全校課表_長表_sample.csv` | 20 位匿名教師、每人約 7 筆課程，共 140 筆 |

---

## CSV 欄位說明

| 欄位 | 說明 | 範例 |
|------|------|------|
| 教師代號 | 字母前綴 + 數字，字母對應組別 | `C01`、`E03`、`N02` |
| 教師姓名 | 匿名化為「科目師XX」 | `國文師01`、`英文師03` |
| 星期 | 1–5（週一到週五） | `2` |
| 節次 | 1–8（第幾節課） | `3` |
| 課程名 | 課程名稱 | `高一國文`、`EnglishA SL` |
| 班級 | 班級代碼 | `101`、`202A` |
| 教室 | 教室代碼 | `T201`、`IB04` |
| 主授科目 | 該老師的主要教授科目（9 類） | `國`、`英`、`自` |
| 細科目 | 更細的科目分類 | `物理`、`化學`、`歷史` |
| 教師類別 | `IB` 或 `普通班` | `普通班` |
| 導師班級 | 導師負責班級（非導師則空白） | `101`、`""` |

## 用範例資料測試腳本

1. 複製 `全校課表_長表_sample.csv` 到 `school_wide/全校課表_長表.csv`
2. 執行：
   ```powershell
   python scripts/audit_categories.py
   python scripts/build_web_school.py
   ```
3. 查看產出的 `school_wide/分類確認表.md` 和 `school_wide/全校代課查詢.html`
